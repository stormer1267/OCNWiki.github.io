Contributing

Issues

Issues

Have you found an issue with the website? That is awesome! Create an issue on our Github Repository and we will get around to it as fast as possible! You can post anything from navigation bugs to spelling mistakes. We want to make sure this project works 100% at any time.

Pull Requests

If you know how to code and you are interested in helping out feel free to submit a pull request on our Github Repository. You can work on fixing an existing issue or fixing an issue you just discovered. Make sure that your fork is up to date before you create a pull request. If the pull request is good we will merge it!
